# xiaohongshu-search Skill

小红书搜索与内容分析Skill，通过MCP接口调用小红书能力，实现搜索、内容读取、评论分析一体化。

## 功能特性
- 🔍 关键词搜索：支持任意关键词的小红书内容搜索
- 📝 内容提取：自动获取帖子完整内容、图片描述、互动数据
- 💬 评论分析：支持加载全部评论和二级回复，分析舆论倾向
- 📊 结构化输出：自动生成包含整体总结、评论分析、参考列表的标准化报告
- 🔗 来源引用：所有内容标注来源，支持追溯原始帖子

## 使用方法

### 触发方式
直接在对话中输入：
```
小红书搜索 [你的关键词]
小红书调研 [你的主题]
小红书总结 [你要分析的内容]
```

### 示例
```
小红书搜索 广州三月展览
小红书调研 2026年春季穿搭
小红书总结 广州美食探店
```

## 输出结构
1. **整体总结**：核心观点、热门分类、争议点，标注引用来源
2. **评论区分析**：意见倾向分布、高频讨论点、典型评价
3. **参考帖子列表**：帖子标题、作者、发表时间、互动数据、笔记ID

## 依赖配置
1. 确保mcporter已安装并运行
2. 已配置Xiaohongshu MCP服务并保持登录状态
3. 无需额外API密钥

## 注意事项
- 搜索结果默认返回前20条，选取点赞量前10条作为分析样本
- 评论默认加载前20条一级评论及二级回复
- 分析结果基于公开用户分享内容，仅供参考
- 遵守小红书平台使用规则，请勿用于商业用途
