# 社交媒体-广告判定

## 使用场景
当分析社交媒体（微博、小红书等）帖子可靠性，需要判断是否为未标注的软性广告时调用本skill。

## 定义
**软性广告**：品牌方委托博主宣传产品的软性广告，通过高影响力博主的带货帖子影响用户的购买决策。帖子中一般不会注明"广告"。

## 广告置信度说明
广告帖子包含的事实性文本、数据可能正确，但通常存在偏向性，例如忽略了竞争对手的优点、掩盖了本品牌产品的缺点；广告帖子中一般会避免提及其他品牌。

帖子符合广告特征数量越多，广告置信度越高。

## 广告帖子特征（符合一个或多个）
| 特征 | 说明 |
|------|------|
| 1. 出现明确的品牌名称 | 帖子中提到了具体品牌 |
| 2. 出现明确的产品名称，并有详细的产品型号 | 具体到产品型号，不是泛泛而谈 |
| 3. 内容仅含有单个品牌 | 全文只讨论一个品牌，没有对比其他同类品牌 |
| 4. 帖子整体评价高度正面 | 全正面评价，没有提及任何缺点或不足 |
| 5. 帖子或者评论区包含前往购物网站的链接 | 提供直接购买链接 |
| 6. 提到与本帖子/博主关联的该品牌折扣 | 比如"通过xxx口令前往xxx平台购买享受折扣" |

## 非广告帖子特征（广告反特征）
符合越多，是广告的可能性越低：
| 特征 | 说明 |
|------|------|
| 1. 高度非结构化的帖子结构 | 例如无分段的文本结构，口语化分享 |
| 2. 未提及任何具体品牌或者产品 | 只是泛泛讨论，没有具体指向 |
| 3. 帖子包含负面情绪或负面评价 | 提及使用过程中的不满或产品缺点 |
| 4. 评论区存在对帖子内容品牌的反向观点 | 其他用户在评论区提出不同意见或负面体验 |
| 5. 帖子属于好物分享，且分享的东西大于两个 |  |

## 判定工作流
1. **提取内容特征**：通读帖子全文，检查是否符合上述广告特征和非广告特征
2. **统计特征数量**：分别统计广告特征命中数和非广告特征命中数
3. **计算置信度**：
   - 广告置信度 = 命中广告特征数 / (命中广告特征数 + 命中非广告特征数)
   - 结果范围：0 ~ 1
   - 置信度 ≥ 0.6 → **大概率为广告**
   - 0.3 < 置信度 < 0.6 → **疑似广告，需结合内容判断**
   - 置信度 ≤ 0.3 → **大概率为非广告**
4. **输出结果**：
   - 给出广告置信度评分
   - 列出命中的广告特征
   - 列出命中的非广告特征
   - 给出最终判定结论
   - 如有需要，给出可靠性建议
